# Song Popularity Prediction (Machine Learning)

## Overview
This project predicts song popularity using audio features with machine learning regression models.

---

## Dataset
- Source: Kaggle (Almost Million Songs Dataset)
- Target: `popularity`
- Features: energy, danceability, acousticness, tempo, valence, etc.

---

## Project Structure
- RQ1.ipynb → Basic regression model
- RQ2.ipynb → Correlation analysis
- RQ3.ipynb → Model comparison
- RQ4.ipynb → Feature engineering
- RQ5.ipynb → Popularity-based trends
- RQ6.ipynb → Data size impact
- RQ7.ipynb → Evaluation metrics

---

---

## How to Run

1. Download the dataset from Kaggle  
2. Open any `.ipynb` file in Jupyter Notebook or Kaggle  
3. Run all cells  
4. Ensure dataset path is correct (`/kaggle/input/...`)
